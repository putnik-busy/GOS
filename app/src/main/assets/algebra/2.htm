Системой линейных уравнений над полем k от неизвестных x1, x2,...,xn называется набор выражений вида
(1) ai1x1 + ai2x2 + … + ainxn = bi  (i=1,2,…,m), где aij,biÎk. Каждое из этих выражений называется линейным уравнением, aij – коэффициентами при неизвестных, bi – свободными коэффициентами уравнений. Решением системы (1) называется строка x0=(x10, x20,…, xn0)Îk, для которой ai1 x10 + ai2 x20 + … + ain xn0 = bi(i=1,2,…,m). Система совместна, если у нее есть хотя бы одно решение. Тогда выражение a1x1 + … + anxn = b (2) называется векторной записью системы (1). Отметим, что x0=(x10, …, xn0) т. тогда является решением системы (1), когда b -линейная комбинация столбцов a1 ,a2 ,...,an с коэффициентами x10 ,x20,...,xn0.

Для системы (1) положим A=(aij)mxn и назовем A матрицей системы (1). Система (1) называется однородной (приведенной), если столбец b ее свободных коэффициентов нулевой, то есть свободный коэффициент каждого из уравнений системы равен 0, иначе – система неоднородная. Однородная система всегда совместна, поскольку строка подходящей длины, составленная из нулей, является ее решением (это решение называется тривиальным решением). Строка (y1(t1,...tp ),..., yn (t1 ,t2 ,...tp )), составленная из n функций yi от переменных t1,…,tp, называется j, общим решением системы (1), если при любых a1,…,apÎk, строка (y1(a1,…, ap),…, yn(a1,…, ap)), составленная из значений ф-ии yi в точке (a1,…, ap) является решением системы (1) и, наоборот, для любого решения x0 = (x10 ,...,xn0 ) системы (1) найдутся a1,…,apÎk, , для которых при всех i = 1,2,...,n имеет место равенство xi0 = yi(a1,…,ap) Таким образом, общее решение является формулой, описывающей все множество решений системы.

Теорема:
а) Множество решений однородной системы от n неизвестных с матрицей ранга r над полем k является подпространством размерности n-r в пространстве kn (это подпространство называется пространством решений системы, а база пространства решений называется фундаментальным набором решений системы).
б) Если x1 ,x2 ,...,xn-r – фундаментальный набор решений однородной системы с матрицей A, то строка Y = t1x1 + … + tn-rxn-r является общим решением этой системы
в) Если (1) – произвольная (неоднородная) система с той же матрицей A и x – одно из ее решений, то строка Y+x -- общее решение системы (1).
